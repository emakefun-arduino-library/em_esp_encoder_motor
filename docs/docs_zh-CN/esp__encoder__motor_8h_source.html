40 * @~English
41 * @class EspEncoderMotor
42 * @brief The main functions of this class are as follows:
43 * @details
44 * -# Supports encoded motors with encoder output signals of A and B phases and a phase difference of 90 degrees
45 * between the signal sequences of A and B channels.
46 * -# Supports driving the motor at a specified speed (RPM) or directly driving the motor with a specified PWM duty cycle.
47 * -# Supports obtaining the current speed information of the motor in RPM.
48 * -# Supports obtaining the encoder pulse count value. This count value is updated at the falling edge of phase A, incremented
49 * by 1 during forward rotation and decremented by 1 during reverse rotation.
50 * -# Supports obtaining the PWM duty cycle currently set on the motor driver.
51 * @example run_speed.ino Rotate the motor at the specified speed (RPM).
52 * @example run_pwm.ino Rotate the motor with the specified PWM duty cycle.
53 * @example forward_stop_backward.ino Move the motor forward, stop it, and then move it backward.
54 * @example detect_phase_relation.ino Connect the encoded motor to the specified positions as described in the example program.
55 * After the program runs successfully, it will print out whether kAPhaseLeads or kBPhaseLeads should be used based on the
56 * actual phase relationship between the A and B phases of the encoder during forward rotation of the motor, helping the user
57 * determine the value that should be set for the phase_relation parameter when creating an EspEncoderMotor object.
58 * @example run_rpm_with_analog_input.ino Dynamically set the rotation speed of the motor according to the analog value of the
